An array is a data structure that holds an ordered list of items. Each slot in a JavaScript array can hold any type of data. An object is a grouping of data and functionality. Data items contained in an object are referred to as properties, and functions on an object are referred to as methods. array does not hold properties. object can hold properties which can be made up of arrays. object can also hold functions knows as method which arrays can not objects hold property/value pairs. a property is a piece of data relevant to the object. Properties are assigned to objects by dot (.) notation or array notation loops allow you to cycle through each element in an array. it's a position in an array starting with the number 0 dot notation uses a dot in the form objectName.property, the bracket notation objectName["property"] dot notation is preferred, bracket notation should only be used when 1. name of property is not a valid identifier or 2. don't know the name of property until run time.
